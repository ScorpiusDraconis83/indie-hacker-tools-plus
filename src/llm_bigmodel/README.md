## 常用 LLM 快速入门模板

### 模板内容简介

本项目旨在提供主流大语言模型（LLM）平台的极简使用模板，当前已收录以下平台的快速调用示例：

- **Gemini**  
- **OpenAI**  
- **Qwen**  
- **Together**  
- **XAI**  
- **ZhipuAI**  
- **文心一言**
